<span class="c5 c15">Getting Started with Artemis</span> {#h.9cr0qpqpazk2 .c10 .c4}
========================================================

<span></span>

<span class="c2">Background</span> {#h.dp5rzvehn2x1 .c4 .c10}
----------------------------------

<span class="c11">Artemis is a genome browser and annotation tool.
Artemis is written in java and can be used on Mac, Linux or Windows.
While it can be used to view any genomic data, it is specifically
designed for viewing bacterial or viral genomes. A variety of file types
can be loaded and viewed using Artemis, these include fasta, multifasta,
EMBL, GenBank and GFF format files.  </span>

<span class="c2">Objectives</span><span class="c11"> </span>

1.  <span>Be able to i</span><span>nstall Artemis on your laptop</span>
2.  <span>Be able to download a bacterial genome sequence</span>
3.  <span>Be able open the genome sequence in Artemis and navigate
    the sequence.</span>

<span class="c2">Section 1: Installing Artemis</span> {#h.djl8ciezoakg .c10 .c4}
-----------------------------------------------------

<span class="c11">Artemis is available at no cost from the Sanger
Institute Website. Instructions for downloading and installing Artemis
can be found at:</span>

<span
class="c0">[http://www.sanger.ac.uk/science/tools/artemis](https://www.google.com/url?q=http://www.sanger.ac.uk/science/tools/artemis&sa=D&ust=1464150347992000&usg=AFQjCNFYe4VM-YIGfoxv3KJtS9Bxq00IYQ){.c14}</span>

<span class="c11">Download and run</span>

<span class="c11">Not working? The most common problem: Java is either
not installed or too old.</span>

<span class="c13">Artemis needs a recent version of Java v1.6 or higher.
Go to java.com and update</span>

<span class="c13">Still not working? Seek help!</span>
